[Test the idle character sprite](https://raw.githubusercontent.com/cmwashum-dot/potatopcti-super-widget-art/main/HostedArt/v5/assets/character/idle/idle_01.webp?v=7.0.0)

[Test the V7 UFO artwork](https://raw.githubusercontent.com/cmwashum-dot/potatopcti-super-widget-art/main/HostedArt/v7/assets/alien/ufo.webp?v=7.0.0)

V7 contains 41 optimized WebP assets. It adds original transparent UFO, antenna, alien-plush, and mystery-crate artwork for the modular Alien Abduction world event. The verified V6.5 character, item, HUD, ground, smoke, and plant artwork remain unchanged. The older couch file is retained only for backward compatibility; V7 does not load or render it.

Keep the repository public and preserve the existing folder structure so StreamElements can load the artwork.
