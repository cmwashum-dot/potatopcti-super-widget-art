# PotatoPcTi_Super Widget Art

Public hosted artwork for the PotatoPcTi_Super StreamElements widget.

## StreamElements base URL

```text
https://raw.githubusercontent.com/cmwashum-dot/potatopcti-super-widget-art/main/HostedArt
```

[Test the idle character sprite](https://raw.githubusercontent.com/cmwashum-dot/potatopcti-super-widget-art/main/HostedArt/v5/assets/character/idle/idle_01.webp?v=6.3.0)

V6.3 contains 37 optimized WebP assets. The walk cycle uses four forward-facing full-body poses with no mirrored footwear, and protected sprite swapping prevents blank action frames. Keep the repository public and preserve the existing folder structure so StreamElements can load the artwork.
