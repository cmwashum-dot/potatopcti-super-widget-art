V7.1 contains 41 optimized WebP assets. Thirteen pickups and eight legacy action poses have been re-extracted from their high-resolution sheets to remove grey edge mattes and baked ground shadows. Their canvas dimensions and gameplay contracts are unchanged. The other 20 assets, including the walk cycle, plant, HUD, ground, smoke, and alien artwork, remain byte-for-byte unchanged.

The older couch file is retained only for backward compatibility; V7.1 does not load or render it. Keep the repository public and preserve the existing folder structure so StreamElements can load the artwork.
